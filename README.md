# Монитор слётов квартир и домов GTA5RP (все сервера)

Скрипт читает [каталог на вики](https://wiki.gta5rp.com/realestate) и шлёт в Telegram при слёте или покупке **квартиры** или **дома**.

## Особенности

- **Все 24 сервера** (Downtown … Senora)
- **Квартиры + дома** на каждом сервере
- Проверка **ровно в :00** каждой минуты (МСК)
- База обновляется при слёте и покупке
- **Ctrl+C** — нормальная остановка

## Настройка

```bash
py -3 -m pip install -r requirements.txt
```

Скопируй `.env.example` → `.env`, укажи токен бота и `chat_id`.  
Открой бота → **/start** → id из [@userinfobot](https://t.me/userinfobot).

```
GTA5RP_SERVER_SIDS=all
```

Только часть серверов: `GTA5RP_SERVER_SIDS=20` или `01,20,24`.

## Запуск

```bash
py -3 murrieta_slet_bot.py
```

Первый полный проход по всем серверам займёт **несколько минут** (24 запроса к вики). Дальше — раз в минуту в :00.

## Важно

- Полный цикл может длиться **2–5 минут** — это нормально для 24 серверов.
- Данные с вики с небольшой задержкой от игры.
